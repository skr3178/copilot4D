# CoPilot4D Agent Configuration

## Python Environment

**Always use the conda environment `nuscenes` for this project.**

To activate:
```bash
conda activate nuscenes
```

This environment should be used for:
- Running Python scripts
- Installing packages
- Running tests
- Any Python-related operations

## Environment Details

- **Type**: Conda
- **Name**: nuscenes
- **Python**: Check with `python --version` after activation
